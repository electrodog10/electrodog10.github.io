Voice Skill Development

Description

  • 15+ Reading Services and Radio Stations

What started as Alexa Skills for radio stations evolved into more complex skills for reading services that serve those with vision disabilities. Working on these projects helped me develop a better ability to create smooth user experiences and interface multiple different APIs together. The skills were primarily developed in NodeJS using the Alexa Skills Kit library.
